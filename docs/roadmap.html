🚀 אבני דרך לפיתוח

ניתן לבצע ב־Sprints קטנים — כל משימה מסתיימת בפעולה ניתנת למדידה.

1) ליבה חזותית

הטמעת עיצוב אחיד בכל הפרקים, התאמת RTL מלאה, בדיקה בדפדפנים (Chrome/Firefox), ושיפור נגישות ARIA.

2) הרחבת תוכן

כתיבת 18 שערים נוספים (7–24), הוספת קלפי תרגול להורדה (PDF/PNG), והמרת כל שער לדף HTML קצר.

3) אינטראקציה

הוספת טיימרים נוספים (90/120 שניות), מונה “אין רק אחת” יומי, ושמירת התקדמות ב־localStorage.

4) פריסה

חיבור GitHub Actions לבדיקת HTML/CSS, דחיפה ל־GitHub Pages, הוספת מטא־דאטה OG/SEO מלאה.

5) תוכן יומי

המרת פזמוני הראפ לקטעי אודיו/ווידאו, יצירת RSS לעדכונים, והטמעת “קלף היום” בדף הבית.

6) גרסת 72 מיקרו־שערים

פיצול כל שער לשלושה מיקרו־תרגולים (תודעה, נשימה, פעולה). כל מיקרו־שער = 90–120 מילים + תרגול קצר.
